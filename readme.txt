== Frequently Asked Questions ==

= How do I install CodexShaper Framework? =

Go to your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter CodexShaper Framework and choose CodexShaper Framework.

Press install -> After installation, click Activate.

= What does mean MVC Framework? =

MVC (Model-View-Controller) is a software design pattern used to separate an application into three interconnected components:

Model: Manages the data and business logic.
View: Handles the user interface and presentation.
Controller: Processes user input and coordinates between the Model and View.

This separation makes the application more modular, scalable, and easier to maintain.
